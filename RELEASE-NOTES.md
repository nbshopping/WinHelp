## 🚀 核心功能

- 🖱️ **顶部热区三键联动**：屏幕顶部左 / 右 / 中键分别绑定文件、浏览器、全局搜索、切歌、音量、任务管理器等
- 🪟 **窗口智能管理**：单击标题栏置顶 / 取消；双击自动缩放居中；底部右侧窗口并排贴靠 / 还原
- ⌨️ **自定义全局快捷键**：自由绑定任意程序，随时开关
- ⚡ **一键系统操作**：关机 / 重启 / 锁屏 / 清空回收站 / 批量关软件
- ✂️ **截图 & 备忘录**：Alt+S 截图，Alt+Z 备忘录
- 🔧 **默认浏览器 & 经典右键**：一键切换默认浏览器，一键回归 Win10 经典右键
- ⏰ **定时关机 / 提醒**：两种互斥模式，到点自动提醒（提前 5 分钟）
- 📘 **内置操作手册**：全功能快捷键一览表

## ⚠️ 关于杀毒软件误报

微软 Defender 等杀毒软件可能对**未签名**的本地程序产生误报，这是正常现象，并非病毒。
如遇误报，请在杀软中选择「**还原**」或「**添加信任**」即可正常使用。

---

## 📝 安装说明

1. 下载对应系统版本（Win10/11 或 Win7）
2. 解压后运行主程序
3. 如需开机启动，在设置中勾选「开机启动」
4. 杀软若误报，请「还原 / 添加信任」
